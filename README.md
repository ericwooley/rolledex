# Rolledex

## Getting Started

### Requirements
docker
node 14+
yarn

1. yarn install
2. yarn dev

You should see lots of console output, including hasura, expo, and the code generator.

Expo should open using [port 19002](http://localhost:19002)
Hasura will open on [port 9695](http://localhost:9695/)
